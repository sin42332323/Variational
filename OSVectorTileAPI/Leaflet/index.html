OS Vector Tile API demonstrator

This sample shows how to display a map using the OS Vector Tile API and Leaflet

To view the map, please enter a valid API key.


To connect to your mapping API:

  1. Go to the OS Data Hub,
  2. Create a project,
  3. Add the OS Vector Tile API to your project
  4. Copy the project API Key, and enter it into the box above.
